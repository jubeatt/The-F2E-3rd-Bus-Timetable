## 總覽

### 關於這份挑戰

這份挑戰是要製作一個台灣公車動態時刻查詢的網站，使用者應該要能夠：

- 在各種裝置上都能夠瀏覽該網站（手機、平板、電腦）
- 透過動態時刻表，來得知公車大約什麼時候到站
- 透過路線時刻表，來查看公車的路線圖
